Why Matic

Decentralized Apps are making huge progress but the current blockchain ecosystem is not prepared to scale as per the demand.

Slow block confirmation and high gas fees need to be soleved before we target mass adoption by mainstream users, And most importantly, it needs awesome user experience.

We aim to change that by simplifying the interaction botween users and the decentralized world. We want to make interacting with the decentralized ecosystem so easy that anyone can do so without worrying about the complexity of the system.

Matic Architecture

Matic Network solves the above problems by building a decentralized platform using an adpted version of Plasma fremwork the provides a soluton for faster and extremely low cost transactions with finality on main chain

The system ensures liveliness using PoS checkpoints which are pushed to the Ethereum. This enables Matic to theoretically achive 2^16 transaction per block on a single plasma chain, and possibly milions os transactions on multiple chains in the future.

Matic Features

scalability

Scalability implemented by using a decentralized Plasma operator mechanism eith finality achieved on main chain

Transaction Throughput

Can scale to milions of transaction on a sidechain tree architecture 65k transactions/second on a single Matic chain

User Experience

Smooth UX abstraction from mainchain to Matic chain. Native mobile apps and SDK with WalletConnect support

Security

The Matic chain(s) operators are themselves Stakers and Delegates in a Proof-of-Stake system in the Matic Network

Assets Interoperability

Assets in different sidechain acrooss various blockchains will be made inter-operable by Matic Network on the near future.
